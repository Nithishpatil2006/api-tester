# K API Tester

A GUI-based API testing tool built with Python and Tkinter. Test GET, POST, PUT, DELETE requests with a user-friendly interface.

## Features
- Test GET, POST, PUT, DELETE requests
- View response status, headers, and body
- Add custom headers and JSON request body
- History of requests
- Dark/Light mode toggle
- Export response functionality
- Clear input/output fields

## Requirements
- Python 3.6+
- requests
- tkinter (usually comes with Python)
